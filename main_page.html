Математические действия

Для арифметических действий ввести путь: /arithmetic/здесь ваше выражение

Пример: /arithmetic/3.1+4


Функционал:

1.Сложение "+"

2.Вычитание "-"

3.Умножение "*"

4.Деление ":"

5.Степень "**"

6.Корень "sqrt"

7.Остаток от деления "!"

8.Целочисленное деление "::"

Для тригонометрических действий с радианами ввести путь: /trigonometric/здесь ваше выражение

Для тригонометрических действий с градусами ввести путь: /trigonometric/deg/здесь ваше выражение

Пример (для радиан): /trigonometric/sin90

Пример (для градусов): /trigonometric/deg/sin90


Функционал:

1.Синус "sin"

2.Арксинус "asin"

3.Косинус "cos"

4.Арккосинус "acos"

5.Тангенс "tg"

6.Арктангенс "atg"
